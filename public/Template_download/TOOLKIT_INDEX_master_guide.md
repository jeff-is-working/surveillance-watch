1. **Choose:** Comprehensive or Simplified template based on needs

### **For Specific Situations**

#### **Situation 1: Agency Won't Respond**
1. Use **Email Template 2** (5-day follow-up) on business day 6
2. Use **Email Template 3** (escalation) on day 10
3. Reference **Troubleshooting section** in How-To Guide
4. Contact ACLU-WA or AG's office if still unresponsive

#### **Situation 2: Denied for Privacy**
1. Read **Legal Context** document section on Skagit County ruling
2. Use **Email Template 4** (privacy appeal)
3. Reference **Troubleshooting Issue 2** in How-To Guide
4. Cite specific court language from Legal Context doc

#### **Situation 3: Agency Says Records Don't Exist**
1. Use **Email Template 7** (records don't exist response)
2. Reference **Troubleshooting Issue 3** in How-To Guide
3. Request documentation of their search

#### **Situation 4: Heavy Redactions**
1. Use **Email Template 9** (redaction justification)
2. Reference **Troubleshooting Issue 6** in How-To Guide
3. Request specific exemption citations

#### **Situation 5: High Fees Quoted**
1. Use **Email Template 6** (fee waiver)
2. Reference **Troubleshooting Issue 4** in How-To Guide
3. Cite public interest factors from Legal Context
